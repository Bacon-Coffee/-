# 日本汉文写本汉方文献用字数据库

## 安装教程

### Windows 安装

> **国内一键下载**（点链接即开始下载，约 284 MB，详见 [下载说明.md](下载说明.md)）：
> - 主链接：`https://ghfast.top/https://github.com/Bacon-Coffee/-/releases/download/v1.0.0/Hanzi-Database-Setup-1.0.0.exe`
> - 备用：`https://gh-proxy.com/https://github.com/Bacon-Coffee/-/releases/download/v1.0.0/Hanzi-Database-Setup-1.0.0.exe`

1. 下载安装包 **`Hanzi-Database-Setup-1.0.0.exe`**
2. 双击运行，进入安装向导
3. 按提示选择安装位置（默认按当前用户安装，**无需管理员权限**），点击「安装」
4. 安装完成后，桌面与开始菜单会自动创建「日本汉文写本汉方文献用字数据库」快捷方式
5. 双击快捷方式启动；首次启动需等待约 30～60 秒完成后端初始化（请耐心等待）

> **首次启动提示**：如出现 Windows Defender SmartScreen 警告，请点击「更多信息」→「仍要运行」。

### macOS 安装

1. 下载 `日本汉文写本汉方文献用字数据库-1.0.0.dmg`（Intel Mac 下载 `x64` 版，Apple Silicon Mac 下载 `arm64` 版）
2. 双击 `.dmg` 文件打开磁盘镜像
3. 将应用图标拖入 `Applications`（应用程序）文件夹
4. 在启动台或应用程序文件夹中双击启动

> **首次启动提示**：如出现「无法验证开发者」提示，请前往「系统设置 → 隐私与安全性」，点击「仍要打开」。

## 功能介绍

本软件为**日本汉文写本汉方文献用字数据库**检索系统，提供以下功能：

### 检索功能

- **多字段组合检索**：支持按字种、字位、语符同时检索，多条件之间为"并且"逻辑
- **服务端全库检索**：检索范围覆盖数据库全部记录（约 1 万条），无遗漏
- **实时分页浏览**：每页显示 20 条结果，支持首页/上一页/下一页/末页快速跳转，并提供页码省略号导航

### 数据展示

- **字帖图片**：每条记录附有字帖图片（支持多张），等比例展示，背景为白色
- **字段信息**：展示字种、字位、语符、文献出处、时代、用法等完整字段
- **时代快速筛选**：点击字段内容中的时代标签，可一键按该时代筛选

## 使用说明

### 启动应用

- 双击图标打开后，屏幕中央会出现「正在启动后端服务，请稍候...」提示框
- 等待约 30～60 秒，提示框消失后检索界面自动打开

### 检索操作

1. 在界面顶部搜索栏中，分别在「字种」「字位」「语符」输入框中输入检索词
2. 点击「检索」按钮（或按 Enter 键）执行检索
3. 检索结果以卡片形式展示，每页 20 条
4. 使用底部分页控件翻页浏览

### 字段说明

| 字段 | 说明           |
| ---- | -------------- |
| 字种 | 汉字的种类分类 |
| 字位 | 汉字本体       |
| 语符 | 对应的语言符号 |
| 文献 | 出处文献名称   |
| 时代 | 文献时代       |
| 用法 | 使用属性       |

## 数据库说明

- 数据库文件存储于系统用户数据目录，软件更新不影响已有数据
- Windows 路径：`%APPDATA%\日本汉文写本汉方文献用字数据库\data.db`
- macOS 路径：`~/Library/Application Support/日本汉文写本汉方文献用字数据库/data.db`
